# TicTT

- Tic tac toe game written in Python

## 📺 Inspiration 
This project was built following the tutorial available here:
[YouTube Tutorial](https://www.youtube.com/watch?v=nbRpDXV7QDM)

## 🎮 Features
- **Graphical User Interface (GUI):** Built using Python's built-in `tkinter` library.
- **Two-player mode:** Play locally against a friend in real time.
- **Clean Design:** Simple grid layout and interactive play mechanics.

## 🚀 How to Run
Ensure you have Python installed on your system. No external dependencies are required!

Once cloned, you can run the game by executing the Python script in your terminal:
```bash
python3 TTT.py
```
